Simple idea: build a pair of command line applications on top of Storage Module API - one for publishing a file to the network, and
another for downloading it. For the scope of this workshop, they could be apps that work on the local machine only.

## Uploader

The uploader publishes a file to the network. To invoke it you would call:

```bash
./uploader <filepath>
```

For this to work locally, you need to set the config option `nat:none`
